# Saarthi Textile Corp – Modern Landing Page

A responsive, animated landing page for **Saarthi Textile Corp**, showcasing premium textile products—including the specialized cloth used for tennis balls. The site features:

- **Custom fabric‑roll animation** in the hero section that reflects the company’s textile focus.
- Product showcase with gradient cards for Tennis Ball Felt, Woolen Blankets, Tweed Fabric, and Endless Felt.
- Testimonials, blog posts, and a Google‑Maps location embed.
- Lead‑capture chatbot integrated with Formspree.
- Dark/Light theme toggle (saved in `localStorage`).
- Fully responsive design built with **React**, **Tailwind CSS**, and **Framer Motion**.
- Deployed on **GitHub Pages**.

🔗 **Live site:** https://saarthitextilecorp.github.io

## Getting Started

```bash
git clone https://github.com/your-username/saarthitextilecorp.git
cd saarthitextilecorp
npm install
npm run dev   # Vite dev server at http://localhost:3000
```

## Deploy to GitHub Pages

1. Push the `main` branch to GitHub.  
2. In the repository **Settings → Pages**, set **Source** to `Deploy from a branch` → `main` / `/root`.  
3. After a minute the site will be live at the URL above.

## Scripts

- `npm run dev` – Start development server.  
- `npm run build` – Build production assets.  
- `npm run preview` – Preview the production build locally.
